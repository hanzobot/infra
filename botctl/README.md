# BOTCTL Config

This directory will define the declarative configuration for BOTCTL instances.

Planned fields:
- instance_name (BOTCTL-1, etc.)
- discord_bot_token (per instance)
- discord_channel (#botributors-test)
- github_pat (read-only)
- memory_path (/var/lib/bot/memory)
- persona (Claude)
- coder (Codex)
